Etiquietas:

<head></head> : todo lo que es informacion de la pagina
<body></body> : todo lo que es contenido de la pagina
Dentro de <head>:
<title>[titulo]</title> : nombre de la pagina, aparece en la pestaña
<link href="[URL]"> : carga un recurso al html? puede ser local o remoto?
Dentro de <body>
<h1>[contenido]</h1> : hace un titulo con [contenido] en negrita. Va de h1 a h6, donde h1 es el mas grande
<p></p> : parrafo
<hr> : linea horizontal
<br> : salto de linea
<label>[etiqueta]</label> : pone un texto como etiqueta al lado de otro objeto
<input> : crea un input. El tipo se establece con el atributo type
<table></table> : crea una tabla
    <thead></thead> : todo lo que se considera encabezado de la tabla, opcional
        <tr></tr> : nueva linea en la tabla
            <td>[contenido]</td> : celda con el contenido
            <th>[contenido]</th> : celda con el contenido en negrita
    <tbody></tbody> : todo lo que se considera cuerpo de la tabla, opcional. Puede llevar las mismas etiquetas que thead adentro
<select></select> : crea un selector de opciones desplegables
    <option>[opcion]</option> : crea una opcion con el texto [opcion] en el desplegable
<button>[texto]</button> : crea un boton con el texto. El tipo se establece con type

Atributos:
name="[name]" : le pone nombre al objeto
id="[id]" : le pone id al objeto
style="[css]" : aplica el estilo escrito en css al objeto
width="[porcentaje]%" : establece el ancho del objeto respecto al ancho de la pagina
align="[lado]" : alinea el objeto a la izquierda (left), derecha (right) o al medio (center)
for="[id]" (label) : al hacer click en la label te lleva al objeto con ese id
type="[tipo]" : (input): establece el tipo de input, como texto, checkbox, etc. (button): button, menu, submit, reset
border="[pixeles]px" : le pone un borde de grosor pixeles al objeto
value="[val]" (option) : le da valor a la opcion? (numerico en selectt?)
